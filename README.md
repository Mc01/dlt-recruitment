# Simple proof of stake

Please read all comments carefully inside CustomToken.sol

## Prerequisites

- Node.JS

## Setup

Install deps
```
npm i
```

## Start local node

Run local node
```
npm run node
```

## Compile

Run compilation
```
npm run compile
```

## Deploy

Run deployment
```
npm run deploy
```

## Test

Run testing suite
```
npm test
```
